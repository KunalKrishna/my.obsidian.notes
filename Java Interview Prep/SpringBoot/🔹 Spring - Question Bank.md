
3 Beginner level SpringBoot interview questions that are served as starters: 

1.How to change the packaging from JAR to WAR in a Spring Boot app?
>Change packaging in pom to war (Maven)
>
2.How to switch from Tomcat server to Jetty server in a Spring Boot app? 
>Exclude spring-boot-starter-tomcat 
>Add `spring-boot-starter-jetty`
>
3.How to handle exceptions globally in a Spring Boot app?
>Create a class with `@ControllerAdvice`. 
>Use `@ExceptionHandler`. 
>Return proper HTTP status and message.
