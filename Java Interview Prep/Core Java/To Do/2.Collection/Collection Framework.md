Java Collection Framework includes :  
- **Interfaces** (`List`, `Set`, `Map`, `Queue` )
- **Implementations** (`ArrayList`, `HashSet`, `HashMap` etc)
	- Abstract : `AbstractMap` 
	- Concrete : `ConcurrentHashMap`, `HashMap`, `LinkedHashMap`, `TreeMap`
- **Utility Classes** (`Collections`, `Arrays`)
